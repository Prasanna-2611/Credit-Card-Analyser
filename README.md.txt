# 💳 CC Analyser — Smart Credit Card Recommender

A full-stack MERN web application that helps Indian users 
find the best credit card based on their income and 
monthly spending habits.

## What it does
- Takes your annual income and monthly expenses as input
- Uses a scoring algorithm to rank 14 real Indian credit cards
- Shows personalised recommendations with direct apply links
- Educates users about credit cards before recommending

## Cards Covered
- Cashback: SBI, Axis, HDFC Millennia, HSBC Live+
- Rewards: HDFC Regalia Gold, Axis Atlas, Amazon Pay ICICI
- Travel: HSBC TravelOne, IDFC WOW, HDFC Millennia, IDFC Diamond
- Premium: HDFC Infinia, IDFC Private, ICICI Emeralde, Axis Magnus
- Student: IDFC WOW Student, Amazon Pay ICICI Student

## Tech Stack
- MongoDB — Database
- Express.js — Backend API
- React.js — Frontend
- Node.js — Server runtime

## How to Run
1. Install Node.js and MongoDB on your PC
2. Clone this repo
3. Run: npm run install-all
4. Start MongoDB
5. Run: npm run dev
6. Open http://localhost:3000
